# Fart Proxy 0.1.0

Burp-like proxy

## Command line options
```
USAGE:
    fart-proxy.exe [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -l, --listen <ADDRESS>    Sets a custom address and port to listen on [default: 127.0.0.1:7878]
```
